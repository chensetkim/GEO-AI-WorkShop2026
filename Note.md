# How to open Colab in Github to google colab? <br>

## Method 1: <br>
The URL Trick (Fastest)Navigate to your notebook on GitHub.Click the browser's address bar.Replace github.com with githubtocolab.com and press Enter.
Example original URL: https://github.comExample modified URL: https://githubtocolab.com
Alternatively, you can manually rewrite the URL prefix:Change https://github.com/ to https://colab.research.google.com/github/.

## Method 2: <br>
From the Google Colab WebsiteGo to Google Colab.Select the GitHub tab in the pop-up window.Paste the GitHub repository URL or type the user/organization name.Choose your repository and branch from the dropdown menu.Click the specific .ipynb file to launch it.(Note: If you need to open a private repository, make sure to check the "Include Private Repos" box and grant the required permissions).

## Method 3: <br>
Use a Browser ExtensionInstall the official Open in Colab Chrome Extension from the Chrome Web Store.Open any notebook page on GitHub.Click the extension icon in your browser toolbar to instantly redirect it to Google Colab.
